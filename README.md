## Javascript Class Example

```javascript
class Queue {
  constructor() {
    this.stack1 = [];
    this.stack2 = [];
  }

  enqueue(input) {
    this.stack1.push(input);
  }

  dequeue() {
    while(this.stack1.length > 0) {
      this.stack2.push(this.stack1.pop());
    }
    return this.stack2.pop();
  }
}
```
